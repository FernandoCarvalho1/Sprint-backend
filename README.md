# API MetaIndústria

API REST desenvolvida com Spring Boot para gerenciamento de sensores industriais.

## Tecnologias
- Java 17
- Spring Boot
- Maven
- H2 Database

## Entidade
Sensor

## Endpoints

POST /sensores
GET /sensores
GET /sensores/{id}
PUT /sensores/{id}
DELETE /sensores/{id}

## Como executar

1. Clonar o projeto
2. Abrir na IDE
3. Executar o projeto (BackendApplication.java)
4. Testar no Postman

## Banco H2
Acessar:
http://localhost:8080/h2-console
